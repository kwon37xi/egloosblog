JPASchemaExport

Hibernate Entity Manager / JPA Annotation 을 사용할 경우 Maven 2에서 Schema Export가 잘 작동하지 않아서 프로그래밍적인 방법으로 해결하였다.

$CLASSPATH/META-INF/persistence.xml 파일이 있다고 가정하고 있다.

 

  1. Ejb3Configuration cfg = new Ejb3Configuration();
    HashMap<String,String> props = new HashMap<String,String>();
    props.put("hibernate.format_sql","true");
    Ejb3Configuration configured = cfg.configure("퍼시스턴스유닛이름을 적는다", props);
    SchemaExport se = new SchemaExport(configured.getHibernateConfiguration());
    se.setDelimiter(";");
    se.create(true, false);

 

이걸로 끝~ 간단하다.

이 글은 스프링노트에서 작성되었습니다.
