[ESB] SonicSoftware의 ESB 정의

ESB의 개념을 처음으로 만든 것으로 알려진 SonicSoftware 에서 Enterprise Service Bus 정의 백서를 내 놓았다.
뭐 관련 기사들을 보니 자사 제품 홍보를 위한 ESB 정의라는 글이 많이 보이긴 하는데, 어쨌든 ESB가 뭔지에 대한 개념이 혼미한 상황에서 ESB의 주창자가 내 놓은 정의이니 읽어볼만은 하겠지.

참고로 관련기사에 링크한 Dave Chappel은 Oreilly 사의 Java Message Service, Enterprise Service Bus, Java Web Services 등의 책을 쓴 Sonic Software의 부사장이자 기술 에반젤리스트이다. 얼마전에 방한했던걸로 아는데, 못봤네.. --;
