[DEBUG] 다운로드 서블릿에 No Cache 코드 사용하지 말것.

파일 다운로드 서블릿에서는 다음과 같이 캐싱을 막는 코드를 넣으면 안된다.
// 캐시 불가
response.setHeader("Cache-Control", "no-cache"); // HTTP 1.1
response.setHeader("Pragma", "no-cache"); // HTTP 1.0
response.setDateHeader("Expires", 0);

IE에서 다운로드해서 저장하는 것은 잘 되는데, 즉시 열기는 안된다. ㅜㅜ
FireFox와 Opera에서는 아무 상관이 없다.
