Hibernate/MySQL 에서 boolean 타입 컬럼 매핑

현재 하이버네이트 3.5에서 자바 boolean 타입은 MySQL "bit(1)"로 매핑하고 있다.

헌데, MySQL 5 부터는 boolean 이라는 타입이 명시적으로 생겼다. http://dev.mysql.com/doc/refman/5.0/en/numeric-type-overview.html

 

이를 일관되게 적용하려면 사용자 정의 Dialect를 만들어서 적용하면 된다.

  1. import java.sql.Types;
  2. import org.hibernate.dialect.MySQL5InnoDBDialect;
  3.  
  4. public class CustomMySql5InnodbDialect extends MySQL5InnoDBDialect {
  5.     public CustomMySql5InnodbDialect() {
  6.         super();
  7.         registerColumnType(Types.BIT, "boolean");
  8.     }
  9. }

 

별거 아닌데 굳이 기록으로 남기는 이유는 대체 왜 boolean 타입을 Types.BOOLEAN 이 아니라 Types.BIT 에 매핑하는지는 몰라서 좀 헤맸기 때문이다.(아직도 모른다.. -- 알려주세요...)

아무튼 이렇게 사용자 정의 타입을 만들어서 Dialect로 등록하면 자바 boolean 타입이 MySQL boolean 타입으로 일괄 매핑된다.
