slf4j Template

slf4j 로거를 자동으로 생성하는 Eclipse 템플릿을 작성해 보았다.

 

  • slf4j : slf4j 클래스를 import하고 로거 객체를 생성한다.
  • debug info warn erro : 각제 로그를 남기는 구문을 자동 생성한다.

 

다른 것은 별거 없는데, 자동 import 방법을 찾는데 좀 시간이 걸렸었다. 헌데, Eclipse 기본 매뉴얼을 찾아보면 다 나오는 것이었다.

 

slf4j.xml 파일을 Preferences -> Java -> Editor -> Templates 에서 import 하면 된다.

이 글은 스프링노트에서 작성되었습니다.
