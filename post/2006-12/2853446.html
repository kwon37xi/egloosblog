Mono : .Net on Linux

Mono는 Microsoft .Net 의 오픈 소스 구현체이다.
리눅스를 비롯한 비 MS-Windows 플랫폼에서 작동한다.

Mono 설치

Fedora Core 3 에서 모노를 설치하려면 모노 공식 사이트 보다는 http://dag.wieers.com/home-made/apt/ 이 리포지토리를 이용하는게 낫다.

/etc/yum.repos.d에 dag.repo 파일을 다음과 같이 생성한다.
[dag]
name=Dag RPM Repository for Fedora Core
baseurl=http://apt.sw.be/fedora/$releasever/en/$basearch/dag
gpgcheck=1
enabled=1

그리고 GPG키를 임포트 한다음에 yum install mono-core 해준다.
$ rpm --import http://dag.wieers.com/packages/RPM-GPG-KEY.dag.txt
$ yum install mono-core


C# 컴파일

모노는 또한 C# 컴파일러도 포함하고 있다. - mcs

$ mcs Hello.cs // 컴파일
$ mono Hello.exe // 실행


자세한 사용법은 다음 링크에 있다.

http://www.gotmono.com/docs/tools/cscomp-cmdlo.html
