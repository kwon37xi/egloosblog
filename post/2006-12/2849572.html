[JS] 변수가 존재하나?

자바 스크립트에서 변수의 존재여부 검사하기.
<SCRIPT language="JavaScript">
var v1="aaa";

if (window.v1) {
    alert("v1이 존재합니다.");
} else {
    alert("v1은 존재하지 않습니다.");
}

if (window.v2) {
    alert("v2가 존재합니다.");
} else {
    alert("v2는 존재하지 않습니다.");
}

if (eval("window.v1")) {
    alert("eval('window.v1')이 작동합니다.");
} else {
    alert("eval('window.v1')이 작동하지 않습니다.");
}
</SCRIPT>

함수의 존재여부도 window.함수명을 이용해서 검사할 수 있다. ()를 주면 안된다.
