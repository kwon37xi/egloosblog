[책] Practical Java

Practical Java

* 초보에서 중급으로 넘어갈때 읽어보자.

Practical Java는 약간 쉬운 수준의 Effective Java라고 봐도 될 것 같다.

사실 프로그램을 짜면서 Effective Java 수준의 주의를 기울여가며 짤 일이 나로써는 흔치 않다.
그에 반해 이 책 Practical Java는 대부분의 초 중급 프로그래머들이 흔히 범하는 실수들을 잘 집어내 준다.
아직 읽고 있는 책이라 정확한 판단은 안되겠지만, 아무튼 나와 내 주위 사람들과 내가 지원나간 싸이트 사람들이 흔히 범하는 실수들이 꽤 많이 지적되어 있는 느낌이다.

번역 상태도 나쁘지 않다.
초급에서 중급 자바 프로그래머로 넘어가기 위해 읽어보면 좋지 않을 까 싶다. 그리고 이 책을 읽고나서 Effective Java도 읽어보고..

헌데... 괜찮은 책 내용과 그런대로 준수한 번역에 비해 편집이 너무 조잡하고 오타도 많이 보인다. --;
정말 싸구려 책처럼 해놨다..
