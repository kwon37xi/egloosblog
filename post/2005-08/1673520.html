[Swing] Swing 어플리케이션용 기본 아이콘 모음

Java look and feel Graphics Repository 에서 자바 Swing 어플리케이션 작성시 사용할 수 있는 기본적인 아이콘 묶음을 받을 수 있다.

무료로 사용할 수 있다고 한다.

각 아이콘을 눌러보면, 아이콘별 역할과 일반적으로 사용되는 단축 글쇠등이 나온다.
되도록이면 다른 개발자들이 개발한 어플리케이션과 일관성을 유지하게 만드는 것이 좋겠다.

별로 이쁘게 꾸미지 않아도 되는 기능 위주의 스윙 어플리케이션을 개발할 때 통일성 있는 아이콘으로써 사용하면 좋을 듯 하다.

이 링크의 홈 디렉토리를 눌러보면 Java Software Human Interface에 대한 글을 볼 수 있다. 일관성있는 인터페이스를 위해 언제 한 번 읽어봐야할 듯 하긴 한데... 언제쯤?? ^^;;

그리고 이 아이콘과 단축 글쇠 사용 예제를 Using the Swing Action Architecture에서 볼 수 있다.

흠.. 요즘 Swing 관련 개발때문에 공부중...
