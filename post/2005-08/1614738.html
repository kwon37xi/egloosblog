Apache Pure Java 관계형 DB - Derby 10.1.1.0 출시

IBM이 Apache에 기증한 Cloud Scape가 Derby 10.1.1.0 으로 아파치 인큐베이터를 떠나서 공식 출시 되었다.

순수 자바 관계형 데이터베이스이고 자바 어플리케이션에 내장되어서도 작동하고 일반 DB 서버처럼 띄워서 접속해서 작동할 수도 있다고 한다.

아마도 현존하는 가장 좋은 순수 자바 RDB 로 알고 있다.

가벼운 어플리케이션을 만들때 DB가 필요하다면 HSQLDB를 내장 DB로 사용하는 것도 괜찮을것 같아 보이는데..

암튼, 관심있게 지켜보는 두가지 순수 자바 DB - Derby와 HSQLDB이다.
