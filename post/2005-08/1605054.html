Ecilpse WTP Snippets에 항목 추가하기

Eclipse 3.1 + Web Tools Project 0.7 에는 Snippets 라는 기능이 있다.

자주 쓰이는 내용들을 속에 값만 채워서 에디터에 삽입하게 해주는 기능인데, 자신의 자주쓰는 항목들을 XML 파일로 Import/Export 해서 다른 개발자의 Eclipse로 복사할 수도 있다.

원래 MyEclipseIDE에 있는건데 그대로 갖다 썼나보다. MyEclipseIDE에 기본 제공되는 Snippets 항목들을 Export 해서 WTP 0.7 에서 Import 했더니 잘 돌아간다.

Snippets 켜기 : Window -> Show view -> Snippets

Snippets 에 항목 XML 파일 Import 하기 : Snippets 창에서 오른쪽 클릭 -> Customize -> Import

MyEclipseIDE의 Snippets들.

나중에 JSTL도 추가해봐야겠다.
