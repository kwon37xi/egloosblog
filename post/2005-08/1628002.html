[UNIX] root 이외의 계정으로는 ftp 로그인이 안돼요!

HP-UX 상에서 작업 중인데, root 외의 계정으로는 로그인이 안된다.

HP-UX 의 FTP 관련 문서를 찾아보고,
/etc/ftpd/ftphosts를 다음과 같이 설정했는데도 불구하고 작동하지 않았다.

allow username myip
allow root myip


그러면서 ftp 클라이언트가 내는 오류라고는, "530 User username access denied..." 가 끝... --;

결론은 내가 사용하는 계정의 셸 때문이었다.

/bin/ksh 를 셸로 사용하는데, Telnet 로그인에는 문제가 없지만 FTP 로그인은 안됐다.

/etc/shells에 다음과 같이 추가해주니 잘 된다.
/bin/sh
/bin/ksh


이 문제는 아마 거의 모든 *nix 계열에 적용될 것 이다. 그리고 root 는 거의 모든 *nix 에서 항상 /bin/sh를 셸로 사용한다.
