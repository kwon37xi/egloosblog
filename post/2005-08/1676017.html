[Swing] Java Swing tips

Java Swing 관련 팁들을 모은다.
◆ 시스템 기본 룩앤필 적용

try {
    // LookAndFeel을 시스템의 기본 값으로 설정한다.
    // Java 5.0에서 Windows XP의 경우 XP 테마로 나온다.
    UIManager.setLookAndFeel(UIManager.getSystemLookAndFeelClassName());
    SwingUtilities.updateComponentTreeUI(JFrame객체); // 모든 UI 변경
} catch (Exception ex) {
    // LookAndFeel 설정 실패
    ex.printStackTrace();
}



◆ 창 최대 상태로 시작

// 창을 최대화한다.
jframe.setExtendedState(JFrame.MAXIMIZED_BOTH);
jframe.setVisible(true);


jframe.pack()을 실행할 필요 없음.

◆ JPanel에 Component를 add 했지만 화면상에 반영이 안된다.
JPanel 객체의 updateUI() 메소드를 실행해야 한다.
