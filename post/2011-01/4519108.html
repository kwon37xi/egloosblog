~ 2011년 1월 3일까지 기술 문서 정리

그닥 크게 정리할 만한 것은 없지만 그래도 해보자.

  • Windows XP를 USB에 깔자 : USB에 ISO 구워서 윈도우를 설치하는게 아니라 윈도우 자체를 USB에 까는 것이다. 새해 들어서 제일 먼저 한 일이다. 이렇게 USB에 윈도우를 깔아 놓고서, 내 노트북의 HDD 파티션에서는 윈도우를 완전히 삭제했다. 이제 내 노트북은 순수하게 Ubuntu Linux만 깔려 있다.
  • Vim 편집 환경 최적화 진행중 : Vim 에디터 편집환경 최적화를 진행중이다. 거의 다 마무리 되었다. 문제는 Vim으로 코딩할 일이 아직 별로 없다는 거? Java 코딩/Unit Test를 Vim으로 하면 미쳐버릴듯.
  • ZenCoding : HTML/CSS 코딩을 CSS Selector를 사용하여 매우 빨리 할 수 있는 에디터 플러그인이다. Vim관련해서 정리하다 알게 되었는데... 엄청나다. 거의 모든 에디터와 IDE를 지원한다.

딱히 눈에 띄는 것은 없지만...
아무튼 기분 좋은 일은 내 노트북 파티션에서 윈도우를 아예 없앤것?
윈도우를 아예 없앨 수 있다면 더 좋을 거 같은데.. 현실은 그렇지 못하다.

게임회사 다니면서 윈도우 없애는 건 좀 이율 배반적인것인데(윈도우 전용 게임으 만들고 있으니까), 게임이 출시 될 때 쯤 되면 게임용으로 SSD에 빵빵한 그래픽 카드를 장착한 데스크탑을 장만할 예정.
