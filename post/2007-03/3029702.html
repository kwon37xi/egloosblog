Ruby/RubyOnRails IDE 비교?

Ruby / Rails IDE Comparison : Idea, Netbeans, RadRails에 루비/루비 온 레일스 IDE 비교가 올라왔다.

얼마전에 공개된 NetBeans 6.0 M7의 Ruby 플러그인이 오픈 소스 제품중에서는 Eclipse의 Ruby 플러그인 보다 낫다는 결과가 나왔다.

Indeed my Ruby / Rails editor of choice for the moment is definitely NetBeans as it’s the one my productivity will increase the most with, and it is free.


그리고 NetBeans 6.0 M8 부터는 나의 숙원(??)이었던 프로젝트 단위로 파일 문자 인코딩 설정 기능이 들어간다.

Windows와 같은 비 UTF-8 계통 운영체제에서 현재의 NetBeans를 사용하면 UTF-8 애플리케이션 작성이 매우 힘들다.
그에 반해 Eclipse는 프로젝트 단위로 문자 인코딩을 지원할 수 있어서, 운영체제의 인코딩에 상관 없이, 다양한 인코딩으로 개발이 가능하다.

우리회사에서는 대부분 Eclipse Ruby/RoR 플러그인을 사용하거나, TextMate(for Mac)라는 프로그램을 사용하는 것 같다.
