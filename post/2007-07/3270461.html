TinyMCE 버튼 목록 위로 놓기

--> TinyMCE 관련 정리 Springnote 페이지
TinyMCE 에디터를 사용하면서 버튼 목록을 위가 아닌 아래에 두었었다.

왜냐하면, 사람들이 글을 쓸 때 제목을 적고 Tab키를 눌러서 바로 편집 영역으로 들어가는데, 만약 버튼들을 편집 영역 위에 두게되면 사용자가 Tab키를 눌렀을 때 그 많은 버튼들을 일일이 다 지나쳐야 편집 영역에 도착해서 사용성이 떨어지기 때문이다.

이렇게 사용성이 떨어지는 것을 해결하는 방법이 생겨서 버튼을 위로 올렸다.
function disableToolbarTabindex() {
    // prototype.js 이용
    $$('.mceToolbarContainer a').each(function(elem) {
        elem.setAttribute('tabindex', '-1')
    })
}

tinyMCE.init({
    language : "ko",
    mode : "exact",
    elements : 'content',
    theme : "advanced",
    //.......
    init_instance_callback : disableToolbarTabindex
});

위와 같이 버튼들의 tabindex를 -1로 지정하면 Tab키를 건너뛴다.
Simple theme는 항상 버튼이 아래에 나오므로 advanced로 변경한다.

원문 : http://www.iamzed.com/2007/06/20/using-jquery-to-skip-the-tinymce-editor-toolbar-when-using-the-tab-key/
원문에서는 prototype.js가 아니라 jQuery를 사용한다.
