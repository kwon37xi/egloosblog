IBM Java Core Analyzer 출시

IBM 에서 Java Core Analyzer 라는 것을 출시했다.

무료로 사용할 수 있는, 자바 코어, 혹은 쓰레드 덤프 분석기이다.

갑자기 JVM이 죽었을 때 JVM은 코어 파일을 떨어뜨린다. 아니면, JVM이 너무 느려졌을 때 왜 느려졌는지 분석하기 위해서 kill -3 JVM의PID으로 쓰레드 덤프 파일을 생성할 수 있다.

문제는, 그 파일이 있어도, 그걸 분석하는게 결코 쉽지 않다는 점이다.

IBM Java Core Analyzer는 그 쓰레드 덤프 분석을 도와주는 도구이다.

근데, 사실 쓰레드 덤프를 떠도 그걸 읽을 줄 모르면 별 소용이 없는데, 그에 관해서는 How to find bottleneck in J2EE application를 참조해서 보면 도움이 될 것 같다.
