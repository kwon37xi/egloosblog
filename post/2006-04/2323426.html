Sun 이 Apache Derby를 Java DB 라는 이름으로 지원한다.

SUN이 Apache Derby를 Java DB라는 이름으로 공식 지원하기로 했다.

거기다가 NetBeans 에서 Derby 기반으로 스윙 데스크탑 어플리케이션 작성하는 예제 강좌도 올ㅤㄹㅣㅆ다.

홈페이지 글을 읽다보니, Derby를 위해 IBM, SUN 등에서 배정된 인원만 50명 정도라는데, Derby에 상당히 신경을 쓰는 듯 보인다.

현재 내가 아는 바로 안정화된 순수 자바 데이터베이스는 HSQLDB와 Derby 정도이다. 둘 다 서버로 띄우거나 어플리케이션에 내장하는 것이 가능하다.

소규모 DB 구축은 HSQLDB로 중대규모는 Derby 만 사용해서, 따로 DB를 띄우지 않고도 자바 어플리케이션 작성하는 것이 가능하다. 사실 Derby 자체도 별로 크지 않기 때문에 Derby 만으로도 Ok~
