[C/C++] Visual C++ 2005 Express Edition 설치하기

본의 아니게 윈도우에서 꼭 Visual C++을 사용해서 C/C++를 해야할 일이 생겨서... ^^;

Visual C++ 2005 Express Edition 은 무료로 사용할 수 있는 MS-Windows 용 C/C++ 개발도구이다.
설치하기

* 참조 URL : VC++ 2005 Express Edition 설치하기

* VC++ 2005 Express 다운로드 에서 VC++ 2005 Express Edition을 다운로드하여 설치한다.
* MS Platform SDK 다운로드 에서 Windows Platform SDK를 자기 PC 플랫폼(보통 x86)에 맞게 다운로드하여 설치한다.
* Tools -> 옵션 -> 프로젝트 및 솔루션 -> VC++ 디렉터리에서 각 항목별로 다음 디렉토리를 추가한다. "Microsoft SDK 디렉토리"는 설치본에 따라 달라진다.
   * 실행파일(Executable files) - C:Program FilesMicrosoft SDK 디렉토리Bin
   * 포함파일(Include files) - C:Program FilesMicrosoft SDK 디렉토리include
   * 라이브러리파일(Library files) - C:Program FilesMicrosoft SDK 디렉토리lib

* Win32 어플리케이션을 작성하기 위해서 %ProgramFiles%Microsoft Visual Studio 8VCVCWizardsAppWizGenericApplicationhtml1033AppSettings.htm 파일에서 다음 부분을 찾아 주석처리(//)하고, VC++ Express 를 재시작한다.
// WIN_APP.disabled = true;
// WIN_APP_LABEL.disabled = true;
// DLL_APP.disabled = true;
// DLL_APP_LABEL.disabled = true;


Win32 프로젝트 생성

   1. File -> New -> 프로젝트 -> Win32 -> Win32 Console Application 선택하고, 프로젝트명 지정.
   2. Application Settings에서 "Precompiled header" 체크 표시 없앰.
   3. Include 디렉토리 지정 : Project -> 속성 -> 구성속성 -> C/C++ -> 추가포함디렉토리 에서 프로젝트에 필요한 Include 디렉토리 지정.
   4. 라이브러리 디렉토리 지정 : Project -> 속성 -> 구성속성 -> 링커 -> 입력 -> 추가종속성 에서 프로젝트에 필요한 라이브러리 파일 지정.
   5. 실행시 명령행 파라미터 지정 : Project -> 속성 -> 구성속성 -> 디버깅 -> 명령인수 에서 프로그램 실행에 필요한 명령행 파라미터 지정.
   6. Build -> 솔루션 빌드 를 통해 컴파일 한다.
   7. 디버그 -> 디버깅시작(F5) 혹은 디버그 -> 디버깅하지 않고 시작(Ctrl-F5)를 통해 프로그램을 실행한다.
