<A> 링크 연속 클릭 방지

<a target='_balnk' href="http:whatever" onClick="return !count++">click</a>

여기서 count 는 어떤 이름이라도 상관없습니다.

단지 다른 JavaScript 전역변수 이름과 충돌하지만 않으면 됩니다.

최초에 저 링크가 클릭되었을 때 count 라는 변수가 생성되고 0으로 초기화가 됩니다. 거기다 !를 했으니 true가 됩니다. 하지만, 그 직후에 count++ 에 의해서 count 가 1이 되겠죠. (count 값은 이미 return 된 이후에 증가합니다.)

그래서 그 다음번에 다시 링크를 클릭하면 return !count++ 에서 count는 1이고, !1 은 false 이므로 false가 리턴되어 링크는 작동하지 않습니다.

위와 같은 원리로 자바 스크립트를 만들어도 연속 클릭을 방지해줍니다.

전역변수 만들고, 모든 폼을 submit 하는 자바 스크립트 함수에서 그 전역변수를 체크합니다. 값이 1이면 return false, 그렇지 않으면 일단 전역변수를 ++ 한 뒤에 return true; 합니다.
