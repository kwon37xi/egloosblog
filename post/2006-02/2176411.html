Java 예외(Exception)가 어디로 간거지??

이런! 예외가 Catch되지 않는다!!!

얼마전 지원나간 싸이트에서 이상하게 예외가 전혀 Catch 되지 않는 사태가 발생했다.

정말 황당했는데, 소스 코드를 보니... 흠...

try {
...
// 여기 어딘가에서 MyException 예외 발생
// 나는 이 예외를 상위 메소드로 넘겨주기를 원한다.
// 그러므로 MyException 에 대해서는 catch를 작성하지 않는다.
} catch (OtherException oe) {
...
} finally {
return result;
}

보다시피 finally 에서 return을 해버렸다.

절대로 어떠한 예외도 상위로 전달되지 않는다!!
이 경우 마치 정상적으로 예외 없이 처리된 듯, 메소드가 정상적으로 리턴되어 버린다.

finally 에서는 return 사용 절대 절대 절대 절대 결코 결코 네버 네버 금지!!
