[UNIX-SH] 셸 스크립트에서 substring 구현하기

셸 스크립트를 작성하는데 사용자가 입력한 문자열을 쪼개서 서로 다른 변수에 저장할 필요가 생겼다.
이럴 때 사용할 수 있는 Unix 명령어는 - expr이다..

expr 자체가 원래 substring 용은 아니고...
암튼, 예제를 보자..

흠.. 정규식을 전혀 모른다면, 쫌 이해하기 그런데...
#셸 환경이라고 가정하고...
# 값을 분리할 변수지정
$ VAR="12345abcde"

# 1. 앞에서 3번째 글자부터 2글자 가져오기
$ expr $VAR : '..(..).*'
34

# 2. 맨 끝에서 3글자 가져오기
$ expr $VAR : '.*(...)$'
cde

# 3. 맨 앞에서 4글자 가져오기
$ expr $VAR : '(....).*'
1234

# 4. 뽑아낸 값을 변수에 저장하기 : Tab 키 위에 있는 `사용.
$ VAR2=`expr $VAR : '...(..).*'`
$ echo $VAR2
45



대충 이해가 갈것 같은데... (...) 이 부분이 바로 실제로 뽑아낼 문자열을 의미한다.
점(.)을 뽑아낼 문자 개수만큼 써주는거다.

1. 맨 앞에서 3번째 글자부터 2글자 가져오기
 보다 시피 맨 앞의 두 글자를 건너 뛰도록 .. 점 두개로 시작하고 그 뒤에 뽑아낼 글자 개수만큼 괄호치고, 그 뒤에 오는 글자들을 모두(*) 무시하도록 .*를 해준다.

2. 맨 끝에서 3글자 가져오기
 맨 앞의 알수 없는 개수의 문자들을 무시하도록 .*으로 시작하고 괄호안에 점 세개로 3글자를 가져오도록 지정한다. 그리고 맨 마지막에서 가져오라는 의미로 $를 붙여준다.

3. 맨 앞에서 4글자 가져오기
 맨 앞에 있는 글자부터 가져오기 때문에 무시할 글자가 없다. 그러므로 바로 괄호로 시작해서 4글자를 지정하고 그 뒤의 모든 글자들을 무시하도록 .*로 마무리 해준다.

4. 뽑아낸 값을 변수에 저장하기 : Tab 키 위에 있는 `사용
 expr 구문을 감싸고 있는 것은 따옴표가 아니라 BackQuote라고 불리는 Tab 키 위에 있는 역 따옴표이다. 이걸 이용하면 어떤 명령의 실행결과 출력을 변수에 저장할 수 있다.

이걸 더 자세히 설명하려면 정규식을 설명해야 하므로 그런건 알아서 공부하면 될것 같고, 저 모양을 보고 따라만 해서 셸 스크립트에서 substring 구현에 아무런 문제도 없을 것 같다.
