[UNIX-SH] 파일에서 줄 끝의 공백을 제거와 기타 Unix 명령행 팁들

파일에서 줄 끝의 공백을 삭제하는 방식으로 VI를 이용할 수도 있고, sed나 awk를 이용할 수도 있다.
평소에는 VI로 하는데, 파일이 너무 클 경우에는 sed로 한다.

VI에서는 :%s/ *$//g 를 이용하면된다.

VI로 열수 없는 큰 파일은 다음 셸 스크립트를 "rmts"라는 이름으로 만들어서 실행하면 된다.

#!/bin/sh
# Remove trailing spaces

# If use did not enter a filename, print usage and exit.
if [ "${1}" = "" ]; then
        echo "Usage : rmts filename"
        exit
fi

mv "${1}" "${1}.rmts.orig"

sed 's/  *$//' "${1}.rmts.orig" > "${1}"
