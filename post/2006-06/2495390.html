[Cygwin] xinetd 설정하고 Windows Service로 올리기

간단하다.

Cygwin Console 에서,
$ xinetd-config
# 뭐라 뭐라 물어보면 그냥 yes

# 다음을 실행하는 순간 Windows의 서비스로 등록된다.
$ cygrunsrv -I xinetd -d "Cygwin Xinetd" -p /usr/sbin/xinetd -e CYGWIN=ntsec

윈도우 제어판의 서비스에서 Cygwin Xinetd를 실행한다.

그리고, 어떤 서비스들을 사용할지는 /etc/xinetd.d 디렉토리에 있는 파일들을 바꿔주면 된다.
보안상 위험하므로, 되도록 사용하지 않는 모든 서비스들을 disable = yes 상태로 바꾼다.

추가 : Windows XP 사용시 서비스를 제공할 포트에 대해 당연히 방화벽에서 예외로 설정해 줘야 한다.
