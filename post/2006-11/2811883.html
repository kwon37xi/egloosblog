[Vim] Plugins I use

내가 사용하는 Vim Plugin 들..

내가 주로 사용하는 Vim 플러그인들을 조금씩 정리한다.
* Linux 등에서 Vim 으로 자바 어플리케이션 작성하기
vtreeexplore : 파일/디렉토리 탐색창

디렉토리 목록을 보면서 파일을 열어서 편집할 수 있게 해준다.
" .vimrc TreeExplore 관련 설정
let treeExplVertical=1
let treeExplHidden=1
let treeExplWinSize=30
map te :VSTreeExplore<CR>

Vim 명령모드에서 te를 누르면 왼쪽에 새로운 창이 생기면서 탐색기가 뜬다.
o를 누르면 해당 디렉토리로 이동하거나 파일이 열린다.
S를 누르면 명령행 셸이 열린다.
기타 도움말은 $VIMRUNTIME/doc/vtreeexplorer.txt 을 참조한다.
