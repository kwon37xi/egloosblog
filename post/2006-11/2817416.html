Windows PowerShell 1.0 출시

Windows PowerShell 1.0 이 정식으로 출시되었다고 한다.
Windows 의 명령행 인터페이스는 Unix 계열에 비하면 너무 짜증이 나고 불편한데, 그에 대해서 MS 에서도 인지를 하고 PowerShell 이라는 것을 만들고 있었다.

정식 출시는 되었다고 하는데 아직 한글판을 다운로드 받을 수는 없었다.

일단 다운로드 페이지는 생성되어 있다 : http://go.microsoft.com/fwlink/?LinkID=75285&clcid=0x09

추가, PowerShell Community Extension 을 설치하면 윈도우 탐색기의 디렉토리에서 오른쪽 마우스 버튼을 클릭하면 Open PowerShell Here 메뉴가 나타난다. 그걸 클릭하면 해당 디렉토리로 이동한 PowerShell 이 실행된다.

추가, 11월 15일 현재 다운로드 가능하다. 간단한 스크린 샷을 첨부한다. MS 역시... 매뉴얼을 벌써 다 우리말로 번역해서 배포했다. 한동안 이거 갖고 놀 수 있을 것 같다.
PowerShell 에서 Vim 등도 전혀 문제 없이 실행된다.
