[DEBUG] Eclipse + Mockito + JUnit 에서 NoSuchField: NULL 예외 발생

Eclipse 3.7 + Mockito 1.9 + JUnit 4.7을 사용하는데 갑자기 NoSuchField: NULL 예외가 발생하기 시작했다.

일단 JUnit 4.10으로 업그레이드를 해보았으나 작동하지 않았다.

계속해서 이유를 탐색하다가 junit-dep 4.4 에 의존성이 걸려 있음을 발견하였다.

NoSuchFieldError when trying to run a jUnit test with Spring 글을 참조해보면 JUnit 4.5 이상을 사용해야 하는데, junit 은 4.7인데, 어떤 라이브러리의 의존성이 junit-dep은 4.4인 이상한 의존관계가 맺어진 상태인 것이다.

불필요한 라이브러리를 삭제하고, 의존관계를 정리하여 JUnit을 완전히 4.10 버전으로 맞췄더니 해결 되었다.
