Stripes - Java 5 annotation 기반 웹 어플리케이션 프레임워크

Java 5 Annotation 기반으로 작동하는 웹 어플리케이션 프레임워크 Stripes 1.01 이 나왔다.

사실 Struts가 강력하고 안정적이지만, 복작한 설정파일과 그 관리의 어려움으로 진입장벽이 상당히 높다. 또한 커스텀 태그의 명칭 등의 일관성이 약간 부족하고 JSTL과 뭔가 엇박자인 것이 사실이다.(이 사항들은 다른 WAF에 비해 상대적으로 그렇다는 것이지 Model 1 방식보다 안좋다는 뜻이 아니다)

Stripes는 완전히 Java 5 기반이며, TSS에 올라온 설명에 의하면 Servlet 2.4/JSP 2.0 기반이라고 한다.

즉, 우리나라에서 가장 많이 쓰일 것으로 개인적으로 예상하는 Java 1.4.2, Tomcat 4.1/Servlet 2.3/JSP 1.2 기반으로는 사용할 수 없다. 하지만 JSP 2.0의 강력한 기능들을 만나고 나면 JSP 1.2로 돌아가고 싶지 않을것이다.

또한 JSTL과 JSF와 어울리게 만들었다고 한다.

요즘, Struts의 대체 경쟁상대 WAF로 Wicket 이라는 것이 상당히 뜨고 있다고 한다. 그와 함께 Stripes의 행보도 궁금해진다.

Stripes는 LGPL 오픈 소스 라이센스를 따른다.
