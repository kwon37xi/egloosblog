Windows XP SP2 IE 6.0 파일 다운로드 클릭시 창 사라짐 현상

설명하기가 좀 애매한데,
Windows XP SP2 IE 6.0에서 어떤 싸이트를 <a href="..." target="_blank"> 이런 식으로 띄우고 새로 뜬 창에서 파일 다운로드를 클릭하면 창이 사라지는 현상이 발생한다.

이건 오직 XP SP2 에서만 발생하는 것으로 IE의 버그가 확실한 것 같다.

어쨌든, 창 사라짐 현상을 없애려면 다운로드 링크에 target="_top"을 해주면 된다.

이거때문에 울회사 사람 고생 많이 했다.. ^^
