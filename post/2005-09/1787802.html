[NetBeans] NetBeans 5.0 베타 출시

NetBeans 5.0 베타가 출시되었다.

원래 4.2로 출시되려 했었으나, 워낙에 막강한 기능들을 많이 추가해서 5.0으로 버전을 바꿔버렸다.

넷빈즈 5.0은 가히 Eclipse의 초강력 도전자라 할 만하다.

특히 웹 개발쪽에서는 사실 이미 4.1 버전부터 Eclipse를 능가한 상태였고.

이번에 아래 글에 올린 이메일 일괄 발송 프로그램을 5.0으로 (일반 Java Application 프로젝트) 개발했는데, 내가 가장 아쉬워하는Alt-Enter를 이용한 힌트(Eclipse의 QuickFix Ctrl-1)가 이제 거의 안정적으로 작동했다. 하지만 뭔가.. 키감이랄까? 손맛이랄까? 그런게 Ctrl-1 과는 달리 좀 이질적이다.

Fix Import는 java.util.* 로 된것을 아직도 클래스 단위로 바꿔주지 못하고 있었다.

어쨌든, 특히 웹 개발자들은 Eclipse + WTP 보다는 넷빈즈가 훨씬 강력하니 꼭 써보라고 권하고 싶다.

혹시 나중에 기회가 된다면 NetBeans 5.0 리뷰를 한 번 써보고 싶다...
