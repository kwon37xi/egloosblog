PostgreSQL SQL Syntax and Use

InformIT에 올라은 PostgreSQL SQL Syntax and Use는 PostgreSQL에 대한 기초적인 설명서이다.

MySQL의 Library 라이센스가 GPL이 되면서 상용 어플리케이션은 MySQL 사용시 비용지불이 필요한 상태가 되었는데, PostgreSQL은 상용 어플리케이션에서도 완전히 자유롭게 사용할 수 있는 몇 안되는 오픈 소스 데이터베이스이다. 이미 몇년 전부터 오랬동안 MySQL과 경쟁상대이기도 하고..

그리고 FireBird 등이 요즘 뜨고 있는 상용 어플리케이션에서도 자유롭게 사용할 수 있는 또 다른 오픈 소스 데이터베이스이다.
