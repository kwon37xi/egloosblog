Java Applet 기반 웹 에디터 Kafenio editor

Kafenio.editor는 자바 애플릿 기반의 웹 에디터이다.

덕분에 자바 애플릿을 지원하는 모든 웹 브라우저에서 사용가능하겠다.

하지만, 스크린 샷과 데모를 봤을 때는 어째 JavaScript 기반의 FCKEditor가 더 나아 보인다. 물론 FCKEditor는 웹 브라우저를 탄다.

속도 문제는 뭐 둘다 비슷한거 같다. 처음 뜨는 속도는 엄청난 양의 자바 스크립트를 로드하는 FCKEditor나 JVM을 띄워야 하는 Kafenio나 느린건 마찬가지.
