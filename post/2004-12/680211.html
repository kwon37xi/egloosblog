[UNIX-SH] *NIX Shell Script 디버깅하기

오늘 셸스크립트 오류 때문에 10여분 헤메다가 일을 끝냈다.

셸 스크립트를 디버깅 하려면, 셸 스크립트의 처리기를 지정하는 부분을 다음과 같이 지정하면 명령이 실행 될 때마다 어떤 명령이 실행되는지 화면에 찍어준다.

$ sh -x 실행할셸스크립트
혹은,
$ sh -v 실행할셸스크립트

-v 옵션이 더욱 자세히 보여준다.

이 이외에도,

$ set -xv

(실행하고자 하는 명령들을 실행한다)

$ set -

위와 같이해도 된다.
