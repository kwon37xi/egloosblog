Java 7 변경된 사항들

Java 7이 출시 되었으나, 우리나라에서는별 반향은 없는것 같다.

사실 Java 5 때 처럼 크게 바뀐점도 없으니까.

게다가 Java 7 Hotspot VM의 loop 최적화 버그 때문에 실제 운영시 사용하면 안되기도 하고.

어쨌든 버그는 시간이 해결해 줄 것인지라, Java 7에서 변경된 점을 정리하였다.
