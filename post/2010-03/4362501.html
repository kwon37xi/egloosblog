Java Web Application Framework - Play

아직 실전에서 쓰일거라 생각하진 않지만 이러한 시도를 했고, 해낸 사람이 있다는 것에 너무도 놀랐다.

Java 코드로 짜는 Ruby On Rails 스타일의 웹 애플리케이션 프레임워크 - Play.
다시 말하지만 Groovy나 JRuby,Jython 그런게 아니다. Java이며, 컴파일과 배포/리로딩을 모두 자동으로 해서 마치 스크립트 언어로 웹 애플리케이션을 작성하는 느낌이 든다.

JPA를 Ruby의 ActiveRecord 스타일로 사용하도록 만들어져 있다.
