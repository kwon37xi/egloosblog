JSP로 System Properties/WAS 정보 모두 출력하기

* 추가 (2006/08/01) : 시스템 프라퍼티 뿐만 아니라 WAS 정보까지 모두 출력하기

단순하지만 가끔 혼란에 빠뜨리는 일 중의 하나...
명령행에서 친 "java -version"과 JSP/서블릿 컨테이너가 실제로 실행되고 있는 JVM이 전혀 다른 경우가 종종 발생한다.
(지원 나간 업체에서 Java 1.3.1에서 실행되는 줄 알고 프로그램을 작성했는데. 1.2로 실행중이어서 문제가 발생한 경험이 있다)
버전이 같더라도 사실 JDK 설치 경로가 다른데, 전혀 다른 경로의 JDK 설정을 바꾸고 왜 안되는거야!! 하고 화낸 경험도 있다.
아래 JSP를 실행하면 해당 컨테이너를 실행시킨 JVM의 System Properties가 출력된다.
매우 단순하지만, 내 경험으로는 리소스와 클래스 위치 찾기 JSP와 함께 간혹 삽질을 일거에 해결해 주는 경우가 생길 수도 있다.
<%@ page contentType="text/plain; charset=euc-kr" %>
Servlet Spec Version : <%= application.getMajorVersion()%>.<%=application.getMinorVersion() %>
Server Info : <%= application.getServerInfo()%>
Context Path : <%= request.getContextPath()%>
Real App Path : <%= application.getRealPath("/") %>
<%System.getProperties().list(new java.io.PrintWriter(out));%>
